# 毒鸡汤小程序

> 微擎后端，开源版本

# 安装

安装好微擎后端模块（代码放在本目录的 `php` 目录里，自行上传到微擎后端 `addons` 目录） 

然后微擎后台，安装好模块，创建小程序平台，获取到 `acid`     
修改小程序代码的 `siteInfo.js`， 修改 `acid` 和 `uniacid`，还有 `siteroot`    

刷新，安装成功
